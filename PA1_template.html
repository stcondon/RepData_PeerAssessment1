Imputation

To find the number of missing values, (which turn out to be a couple of days missing from the dataset), we find the number of true values in the vector we used to eliminate the NAs in the first place. After that, we have to reimport the dataset, since earlier we excised the missing data. I chose to impute values from the average daily activity pattern we found above for the missing values.

sum(bad)
## [1] 2304
activity <- read.csv('data/activity.csv')
## if anyone knows a more effective way to do this part, please let me know
for(row in 1:nrow(activity)) {
       if(bad[row] == TRUE) {
              activity$steps[row]<-df$steps[df$interval==activity$interval[row]]
       }
}

Time for the Weekend!

Now, to distinguish weekend from weekday, we add a category onto the activity frame, and then use it to create our plot.

weekday <- vector()
for(row in 1:nrow(activity)) {
       if(isWeekday(activity$date[row]) == TRUE) {
              weekday[row] <- "Weekday"
       } else {
              weekday[row] <- "Weekend"
       }
}
activity$Weekday <- weekday
df <- ddply(activity[,-c(2,4)], .(weekday, interval), colwise(mean))
qplot(interval, steps, data = df, xlab = "Time of Day", geom = "line", facets =  weekday ~ .)
